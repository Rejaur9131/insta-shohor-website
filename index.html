Let's answer some questions!!!

How does javascript work?

A browser first loads HTML & CSS to create a web pages by the DOM(Document Object Model). Then the browser loads the JavaScript to execute in the order the code is written.When browser gets javascript code, it goes through V8 engine in chrome and then converted to machine code using turbo fan and shows output.

What does javascript event loop do?

JavaScript is a single threaded language and combination of synchronous & asynchronous events. When it gets multiple events, it creates a queue of events. After finishing tasks of callStack it triggers event queue and executes event one by one until finishes. This like is a loop that's why it is called event loop.In short, Event loop triggers the events of event queue for execution.

What are the differences between local storage vs session storage ?

Local storage and session storage are same in terms of whether they can be sent with requests, and differences are:

  • Local Storage: Capacity:10mb, accessible from:Any window, expires:never, storage location:Browser only
  • Local Storage: Capacity:5mb, accessible from:Same Tab, expires:on tab close, storage location:Browser only
